# cors-tests
Testing how CORS works

## With NodeJS

### Prerequisites
- nodejs (I have v6.3.1)
- 'npm install express'

### How to start

- 'cd withNodeJs'
- 'node server.js'
- In a browser: 
    - http://localhost:8081/listUsers  (list all users)
    - http://localhost:8081/3  (get details of a user by ID)
